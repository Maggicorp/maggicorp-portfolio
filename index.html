The Print Store

A collaboratively developed eCommerce app which uses the third party API stripe.
Print shop is an ecommerce app which was developed collaboratively. Git and GitHub were used for version control. The app uses Stripe for payment processing, as is deployed in test mode. The front end was developed using Express, Handlebars and Javascript, and the back end uses MongoDB and Mongoose.
GitHub repository for the Front End
GitHub repository for the Back End

Advice App

A full stack rails application that gives you advice and allows you to give yourself advice.
The Advice App is a full stack ruby on rails application. The rails back end is deployed on Heroku. The app uses two third party api's, one to generate user names, and one to offer example advice. The front end is written in javascript and uses JQuery and Bootstrap.
GitHub repository for the Front End
GitHub repository for the Back End

Tic Tac Toebot

A tic tac toe game developed using javascript.
Tic Tac Toebot is a tic tac toe game. The front end game logic is written in javascript and designed in html and css. The back end is a prewritten database for tic tac toe. The game correctly calculates game outcomes and retrieves relevent game data for both individual games and all games.
GitHub repository for the front end only
